아이폰 사파리 브라우저 [홈 화면 바로가기] 만들기 안내

1. 브라우저 “사파리”를 실행합니다.

2. 사파리 브라우저 하단 메뉴의 “더보기” 버튼을 선택합니다.

3. 목록에서 “홈 화면에 추가”를 선택 합니다.

4. 홈 화면에서 보여질 이름을 설정하고 오른쪽 상단의 “추가” 버튼을 선택 합니다.

5. 4 에서 설정한 이름인 “AI RINGO”로 생성된 것을 확인할 수 있습니다.

아이폰 크롬 브라우저 [홈 화면 바로가기] 만들기 안내

1. 브라우저 “크롬”을 실행합니다.

2. 크롬 브라우저 상단 우측의 “더보기” 버튼을 선택합니다.

3. 목록에서 “홈 화면에 추가”를 선택합니다.

4. 홈 화면에서 보여질 이름을 설정하고 오른쪽 상단의 “추가” 버튼을 선택 합니다.

5. 4 에서 설정한 이름인 “AI RINGO”로 생성된 것을 확인할 수 있습니다.

PWA 만들기 안내

2. 목록에서 “홈 화면에 추가”를 선택 합니다.

3. 홈 화면에서 보여질 이름을 설정하고 오른쪽 상단의 “추가”를 선택 합니다.

4. 3 에서 설정한 이름인 “AI RINGO”로 생성된 PWA를 확인할 수 있습니다.

크롬 Pushl알림 받기 설정 안내

1. 브라우저 “크롬”을 실행합니다.

2. 로그인 후 크롬 브라우저 우측 상단의 더보기(⋮) 버튼을 선택하고 “설정”을 선택합니다.

3. 설정 화면 상단 좌측 “햄버거 메뉴”를 선택합니다.

4. “보안 및 개인정보 보호” 선택 합니다.

5. 개인정보보호 및 보안에서 “사이트설정” 선택 합니다.

6. 권한에서 “알림설정” 선택 합니다.

7. 기본 동작에서 “사이트에서 알림 전송 허용을 요청할 수 있음” 선택 하면 Push알림을 받을 수 있습니다.

8. 사이트별 알림 설정은 하단 맞춤 설정된 동작에서 주가 버튼을 선택하여 *.ringo.co.kr URL입력합니다.
